# Freqtrade Configuration Files

This directory contains Freqtrade configuration templates and generated configs.

## Files
- `config.backtest.example.json` - Template for backtesting configuration
- `config.dryrun.example.json` - Template for dry-run configuration
- `config.generated.example.json` - Template for generated configuration

## Purpose
HER generates Freqtrade configs dynamically based on run parameters. These templates serve as the base for configuration generation.
